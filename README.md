# EpiBioStat Studio

Plataforma educacional interativa de Epidemiologia e Bioestatística para pesquisas em saúde.

## Versão

EpiBioStat Studio 7.2

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub chamado `epibiostat-studio`.
2. Envie todos os arquivos desta pasta para o repositório.
3. No GitHub, acesse **Settings > Pages**.
4. Em **Build and deployment**, selecione:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
5. Clique em **Save**.
6. O GitHub vai gerar um link parecido com:
   `https://SEU-USUARIO.github.io/epibiostat-studio/`

## Arquivos principais

- `index.html`: página inicial do site.
- `plataforma.html`: EpiBioStat Studio 7.2 completo.
- `versoes/epibiostat-studio-7-2.html`: cópia da versão atual.
- `.nojekyll`: evita processamento Jekyll no GitHub Pages.

## Observação

O módulo de cálculo amostral é didático. Para pesquisa real, confirme os resultados em softwares específicos e com apoio de bioestatístico.
